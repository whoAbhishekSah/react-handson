Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

#### Build tools

1. Formatting check: `npm run fmt`
2. Formatting Fix: `npm run fmt-fix`
3. Lint check: `npm run lint`
4. Lint Fix: `npm run fix`
